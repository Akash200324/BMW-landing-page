The "Glass Box" Module: A centralized, large, fixed rectangular panel with rounded corners must be the focal point, emulating the transparent HUD (Heads-Up Display) overlay style seen in your reference images. This module must use "Glassmorphism"—a semi-transparent backdrop with extreme background-blur (backdrop-blur-xl bg-white/5 border border-white/10).

Layout Structure: This main module is a flexbox container, split 40/60:

Left Feature Menu: This vertical section lists the six main feature categories (Powertrain, Design, Tech, Luxury, Charging, i7 Specs).

Right Dynamic Content: This larger section dynamically updates based on the selected feature, displaying a large descriptive title, an iconic image or animation of that feature, and a "Read More" button.

Fluid Slide Transitions: The entire feature section must sit atop an automated background image slider. The slider should smoothly transition between your four chosen background images (the front fascia, the profile, the mountain shot, and the moonlight scene) every 8 seconds.

3. Interactive Left Feature Menu (Detailed List)

Listing Format: List the categories as a vertical stack of interactive menu items. Each item must consist of a unique vector icon (matching the clean, geometric style of your reference images) and a concise, uppercase title, such as "POWERTRAIN," "DESIGN," "TECH," "LUXURY," "CHARGING," and "I7 SPECS".

Hover Interaction (Vibecode Style):

When a user hovers over a menu item, it must illuminate with a soft, pulsing electric-blue accent line or outer glow (text-blue-500 or border-b-2 border-blue-500).

An interaction must occur in the Right Content Section: the description, icon, and title must simultaneously update with a smooth, staggered animation, instantly providing information about that feature (e.g., "POWERTRAIN" shows performance specs; "TECH" shows iDrive screen information; "CHARGING" shows battery details).

Use minimal, outlined icons and large, precise data numbers, formatted as "449 HP," "4.6 SEC," and "728 KM WLTP" for the i7 models.

Micro-Animations: On hover, these specific data points could have a subtle, digital "climbing number" effect or a subtle geometric rotation of the icon.

Framer Motion Integration: Ensure all transitions (feature swiping, data point updates, image fade-ins) are powered by smooth, staggered Framer Motion animations to give the interface a fluid, premium feel.

"Vibecode" Vibe: All text should use modern, sans-serif typography (like 'Avenir Next' or 'Satoshi'). Use uppercase for titles and clean, outlined buttons (like 'GO TO SHOP' in the reference images) with a gentle blur effect.
